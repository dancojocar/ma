LocaleRecyclerView - Solution code
==================================

The localized version of the RecyclerView starter app.

Introduction
------------

This app uses a RecyclerView to display data. It provides a
custom adapter for the RecyclerView. This version has been
localized for French and Hebrew to demonstrate how to
find and extract string resources for localization.

Pre-requisites
--------------

You need to know:
- How to open, build, and run apps with Android Studio.
- How to switch languages on an Android device or emulator.
- How to extract string resources and use string resources in the code.
- How to add languages and edit translations in the Translations Editor.

Getting Started
---------------

1. Download and run the app.

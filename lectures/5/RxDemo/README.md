# RxJava Demo

This is a simple Android app to demonstrate the usage of RxJava.

## Features

*   **HelloWorld**: A basic example of an `Observable` and `Observer`.
*   **Operators**: Demonstrates the usage of `map`, `skip`, and `filter` operators.
*   **Fetch**: Shows how to use `zip` to combine the results of two `Observable`s that simulate network calls.
*   **Debounce**: An example of how to use the `debounce` operator to handle user input.

## How to Run

1.  Clone the repository.
2.  Open the project in Android Studio.
3.  Run the app on an emulator or a physical device.
